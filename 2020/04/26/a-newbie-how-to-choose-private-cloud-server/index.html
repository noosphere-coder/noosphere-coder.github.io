新手如何选择云服务器

2020-04-26 开发环境 云服务器 云主机 阿里云 腾讯云 华为云 阅读次数 0

文章目录

  1. 主要考虑因素
  2. 充分利用好新人身份
  3. 买1年还是3年的套餐

最近在个人小项目需要购买一些服务器,花了一些时间,对市面上主要的云服务提供商做了一些简单的对比,记录下来备忘,同时,也给希望可以给同样需要的人提供一些思路,节省时间.

主要考虑因素

云服务器商的各种套餐让人眼花缭乱，原因是因为各自的资源性能不一样。

虽然有一些标准可以对比，根据个人的使用经验，但是实际性和云服务商的平台架构和技术实力还是差别蛮大，比如不同平台，物理隔离的技术的差异，可能你的服务器会因为隔壁大户老吴抢资源太厉害，而遭遇性能问题。

从不了解到知道再到熟悉，有一个过程。所以对于新手来讲，先入门体验一下是最重要的，而搞清楚服务商之间的种种差别反而不太重要，所以，初次体验，我建议成本怎么低怎么来。

现在基本上各个服务商都有针对新人的优惠，如下面是阿里云，腾讯云，华为云这三个大服务商现在疫情期间搞的对新人的活动价格对比表：


注: 上面这个是之前我买服务器时候的价格对比,现在价格已经发生变化了,这个表格主要表达的是对于新手来讲,价格差别不是特别大,至于新的价格,可以上面的连接进去看看最新的就好了

可以看到，从价格上来讲，大家的价格差别不是很大，其实选哪个都差不了太多，其中阿里云对于新手入门级的价格可以说是非常优惠，腾讯云99元是因为它的硬盘存储是50G,我上面没有列出来。

充分利用好新人身份

云服务商都想套死用户,所以他们提供给新人的优惠特别大,比如上面这种优惠,老用户是不可能获得的,老用户如果过期了,想再去买,那要贵好几倍.., 所以,我建议第一次购买,如果手头宽松,可以尽量买好一点的,而且时间买尽可能长的.

那么如果你已经买了,新身份过期(就是老用户了),怎么办呢? 我建议

  1. 换一个服务器提供商，用新人身份去买,还是同样的策略,尽量买更长的时间,手头宽松就买尽量好的配置
  2. 等优惠券或者搞活动的时候一次性买几年的套餐

其中,换服务商的唯一问题是域名指向迁移,其实也不复杂,因为域名原来都已经备案通过了,迁移是没有问题的.

买1年还是3年的套餐

以下是当时的信息，已经变化,请以实际为准

如果你点进去的话，会看到他们都会有批量购买3年的套餐，还会折上折。

我个人推荐一次买3年的那种套餐，因为很多时候买了服务器，往往可能开始玩一段时间就会放下，很有可能等到下一次玩就是一年之后了，这个时候你已经不是新人，再买成本就高了。

比如你买了一个服务器后，最开始玩可能是搭建一个博客，一年后，可能突然你想写个小程序，需要一个api服务端。这个时候如果你之前买了3年的套餐，可以继续使用而不需要重新去购买。

目前三个服务商最低配置(1和2G),三年低至：233.29元
阿里云 1核CPU 2G 内存 1M 带宽 40G存储的配置， 3年套餐233.29元

腾讯云 1核CPU 2G 内存 1M 带宽 50G存储的配置, 3年套餐299元

华为云 1核CPU 2G 内存 1M 带宽 40G存储的配置, 3年套餐 236.65元

从价格上最低配置，腾讯云稍贵一点,因为多10G存储，其实一般来讲，入门玩玩的目的，阿里云的40G也够的了，价格也低，不差那点钱的和有特殊要求可以考虑腾讯云。

好了，以上是对新手入门选择服务器的建议，其实没有什么特别需要考虑的，因为各大服务商提供给新手的选择都差不多，此文主要给大家一个思路，就是新手入门没必要花太多时间在各个平台商的对比上，另外一个，新手一次性买3年很重要

挖个坑，下次有时间按场景来写写服务器选择。
